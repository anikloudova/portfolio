# portfolio
### Popis projektu
Toto portfolio prezentuje mou práci v oboru Multimediální komunikace. Obsahuje tři hlavní webové projekty (Statek Kersko, Psí útulek, The Lunaz Prague) a ukázky grafické tvorby v Adobe nástrojích.

### Použité technologie
* **Front-end:** HTML5, CSS3 (Flexbox, Media Queries), JavaScript, PHP.
* **Design:** Adobe Illustrator (loga), InDesign (sazba), Photoshop (úprava fotek).
* **Hosting:** GitHub Pages.

### Postup nasazení
Web je nasazen pomocí služby **GitHub Pages**. Zdrojové kódy jsou uloženy ve větvi `main`. Nasazení probíhá automaticky z kořenové složky repozitáře. HTTPS je aktivní.

### Seznam zdrojů a licencí (Bod 5)
* **Vlastní tvorba:** Všechny texty, grafika Argo Bags, autoportrét a kód.
* **Školní projekty:** Fotografie pro Statek Kersko a Psí útulek.
* **Ikony:** Font Awesome / vlastní vektory.

### Sledování návštěvnosti
Zvolena **Varianta B – čistě PHP**. Loguji počet zobrazení do souboru `counter.txt`. 
*Poznámka: Na živém webu GitHub Pages není PHP podporováno, kód je součástí ZIP archivu.*
